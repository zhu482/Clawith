# 04 - 标题生成

## 职责定位

根据文章内容类型，智能选择标题流派。

**核心问题**：什么标题能吸引目标读者？

---

## 调度逻辑

### 场景判断

**步骤1：分析文章内容**
- 读取 `03_初稿.md`
- 提取核心主题
- 判断内容类型

**步骤2：选择流派**

| 内容类型 | 目标受众 | 调用文件 |
|---------|---------|---------|
| **科技产品类** | 科技从业者、产品经理、开发者 | `04-标题生成-AI圈.md` |
| 工具测评、技术分析、产品推荐、效率提升 | | |
| **个人成长类** | 普通人、职场人、创业者 | `04-标题生成-知识付费.md` |
| 赚钱方法、认知升级、能力提升、方法论 | | |
| **混合场景** | 用户要求"多种风格" | 两个都调用，各5个 |

---

## 使用方式

### 自动判断（推荐）

```
系统：分析文章内容
→ 判断：科技产品类
→ 调用：04-标题生成-AI圈.md
→ 输出：10个候选标题
```

### 用户指定

| 用户说... | 调用文件 |
|----------|---------|
| "用AI圈风格起标题" | `04-标题生成-AI圈.md` |
| "用卡兹克风格起标题" | `04-标题生成-AI圈.md` |
| "用知识付费风格起标题" | `04-标题生成-知识付费.md` |
| "用小冯风格起标题" | `04-标题生成-知识付费.md` |
| "给我多种风格的标题" | 两个都调用 |

---

## 输出

- 10个候选标题
- 标注每个标题的流派
- 推荐指数（⭐）
- 适用场景说明

---

## 质量标准

### 冠军检查清单
1. ✅ 情绪浓度：是否有情绪词？
2. ✅ 数字颗粒度：数字是否具体？
3. ✅ 对象具体化：是否用了具体名词？
4. ✅ 主角感：是否让读者觉得"这跟我有关"？

### 红线审查
- [ ] 承诺了完全做不到的事
- [ ] 纯制造焦虑但没有解决方案
- [ ] 夸张到不可信的程度
- [ ] 违反平台规则或价值观
