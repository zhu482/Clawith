# 飞书文档排版

## 排版风格

**清爽简洁风**

### 配色方案
- **主色**：`#1F2329` (深灰黑)
- **强调色**：`#3370FF` (飞书蓝)
- **背景色**：`#FFFFFF` (白)
- **灰色**：`#646A73` (辅助文字)
- **引用背景**：`#F7F8FA` (极浅灰)

### 标题样式
- **一级标题**：左对齐 + 底部细线
- **二级标题**：左侧蓝色竖线 + 稍大字号
- **三级标题**：加粗 + 蓝色小圆点

### 特殊处理
- 加粗文字：`font-weight: 600`
- 引用块：浅灰背景，左侧蓝色细线
- 代码块：保持等宽字体

---

## 使用方法

### 基础用法
```bash
python3 .agent/skills/01-内容创作/长文/scripts/format_article.py \
  "/path/to/article.md" \
  --platform feishu
```

### 指定输出路径
```bash
python3 .agent/skills-v2/01-内容创作/排版/scripts/format_article.py \
  "/path/to/input.md" \
  --platform feishu \
  --output "/path/to/output.html"
```

---

## 交付物

脚本运行后，会在源文件同级目录下生成 `*_FEISHU.html` 文件。

### 使用步骤
1. 双击打开生成的 HTML 文件
2. **Cmd+A** (全选)
3. **Cmd+C** (复制)
4. 去飞书文档粘贴

---

## 技术细节

### 图片处理
- **保留路径**：Markdown 中的图片路径保持不变
- **本地引用**：飞书支持本地图片上传
- **需手动上传**：粘贴后可能需要重新上传图片

### CSS 内联
- 所有样式内联到 HTML 标签中
- 飞书对部分 CSS 支持有限
- 建议粘贴后检查格式

### 智能清洗
- **段落合并**：自动识别 Markdown 换行
- **代码块保留**：等宽字体 + 浅灰背景
- **表格优化**：自动添加边框和间距

| 特性 | 微信公众号 | 飞书文档 |
|------|-----------|---------|
| 强调色 | 深红 `#ab1942` | 飞书蓝 `#3370FF` |

## 注意事项

1. **图片上传**：飞书粘贴后可能需要重新上传图片
2. **格式检查**：飞书对 HTML 支持有限，粘贴后检查格式
3. **字体支持**：飞书默认使用系统字体
4. **表格优化**：复杂表格建议使用飞书原生表格功能

---

## 常见问题

### Q: 粘贴后图片不显示？
A: 飞书文档粘贴后需要重新上传本地图片，或使用在线图片链接。

### Q: 格式部分丢失？
A: 飞书对某些 CSS 属性支持有限，建议粘贴后手动调整。

### Q: 代码块没有高亮？
A: 飞书文档不支持代码高亮，建议使用飞书原生代码块功能。

### Q: 能否导出为 PDF？
A: 可以，飞书文档支持导出为 PDF，格式会保留。
