# 口播稿-检查

> **发布前质检，确保爆款三指标**

## 零、第一原则

**真诚度是一票否决项。**

如果真诚度检查不通过，其他指标再高也不建议发布。

---

## 一、真诚度检查（一票否决）

### Q1: 这像朋友深夜聊天吗？

**判断标准**：
- ✅ 用"你"而不是"大家"
- ✅ 有自我暴露（失败经历、真实情绪）
- ✅ 口语化表达（疑问句、强化副词）
- ❌ 说教口吻
- ❌ 过于正式的书面语
- ❌ 像老师讲课

### Q2: 有没有为了技巧而变得刻意？

**判断标准**：
- ✅ 金句间隔合理（20-30秒）
- ✅ 节奏自然流畅
- ✅ 故事真实可信
- ❌ 10秒内连续3条金句
- ❌ 为了金句而金句
- ❌ 过度煽情

### Q3: 我自己会转发这条视频吗？

**判断标准**：
- ✅ 有启发
- ✅ 有实用性
- ✅ 能代表我的态度/品味
- ❌ 内容空洞
- ❌ 纯粹追热点
- ❌ 不符合我的价值观

**输出结论**：
```markdown
## ⚠️ 一票否决项：真诚度检查

□ Q1: 这像朋友深夜聊天吗？ ✅通过 / ❌不通过
□ Q2: 有没有为了技巧而变得刻意？ ✅无 / ❌有
□ Q3: 我自己会转发这条视频吗？ ✅会 / ❌不会

👉 结论：✅通过 / ❌不通过（需重写）

【如果不通过，停止检查，要求返回修改】
```

---

## 底层规律自检（六大机制）

**在检查具体指标之前，先检查底层机制是否到位**

### 机制1：认知缺口管理
**原理**：人脑讨厌"未完成的认知"

**自检**：
- [ ] 观众看完第一段，会想看第二段吗？
- [ ] 有没有"暗示后面更牛逼"的设计？
- [ ] 每个段落结尾有没有留悬念？

**不通过 → 完播率<50%**

---

### 机制2：预期违背
**原理**：所有情绪来自"预期"与"现实"的落差

**自检**：
- [ ] 有没有"你以为X，其实Y"的设计？
- [ ] 有没有"身份/观点/起手牌"的反差？
- [ ] 钩子是否打破了观众的常规预期？

**不通过 → 开场流失严重**

---

### 机制3：认知爽点密度
**原理**：短视频的密度是"认知爽点"，不是知识点

**自检**：
- [ ] 有没有连续3段都在讲道理？
- [ ] 每30秒有没有"新信息/新故事/新金句"？
- [ ] 是否包含顿悟/共鸣/获得三种爽点？

**不通过 → 中段流失严重**

---

### 机制4：具象化认知
**原理**：大脑不擅长处理抽象，但擅长记忆画面

**自检**：
- [ ] 有没有"本质"、"认知"、"思维"这些抽象词？
- [ ] 能不能用"有画面感"的比喻替代？
- [ ] 故事是否有足够的细节和画面感？

**不通过 → 听不懂/记不住**

---

### 机制5：社交货币设计
**原理**：用户分享是为了给自己贴标签

**自检**：
- [ ] 这句话发朋友圈，会显得我很low吗？
- [ ] 这句话能帮用户立什么人设？
- [ ] 满足"显得有深度/上进/清醒/乐于助人"中的至少1个吗？

**不通过 → 分享率<1%**

---

### 机制6：三道心门击穿
**原理**：必须依次打开观众的三道心门

**自检**：
- [ ] 第一道（凭什么听你的）：有没有权威借用/数据支撑？
- [ ] 第二道（跟我什么关系）：有没有起手牌理论/同类人故事？
- [ ] 第三道（看完有什么用）：有没有阶梯推力/实用方法？

**不通过 → 整体数据偏低**

---

**快速诊断表**：

| 问题症状 | 可能原因 | 对应机制 |
|---------|---------|---------|
| 开场流失>70% | 钩子无力 | 机制2：预期违背 |
| 中段流失严重 | 节奏太慢 | 机制1：认知缺口 + 机制3：爽点密度 |
| 完播率正常但点赞率低 | 没有金句/共鸣不够 | 机制4：具象化 + 机制5：社交货币 |
| 分享率<1% | 不够实用/没有传播点 | 机制5：社交货币 |
| 整体数据偏低 | 信任感不足 | 机制6：三道心门 |

## 二、完播率检查（60%权重）

### 完播率分级诊断

**快速定位问题区间**：

| 完播率 | 等级 | 核心问题 | 优先修复 |
|--------|------|---------|---------|
| <30% | D级 | 开场失败 | 前3秒钩子（预期违背） |
| 30-50% | C级 | 节奏太慢 | 认知缺口管理（推力设计） |
| 50-70% | B级 | 密度不够 | 认知爽点密度（每30秒1个） |
| 70-85% | A级 | 优秀水平 | 微调优化即可 |
| >85% | S级 | 爆款潜质 | 保持+放大优势 |

**诊断流程**：
```
1. 看完播率区间 → 定位等级
2. 看核心问题 → 找到病因
3. 看优先修复 → 应用对应机制
```

**示例**：
```
完播率45% → C级 → 节奏太慢
→ 优先修复：检查是否有"阶梯推力"设计
→ 自检：观众看完第1段会想看第2段吗？
```

---

### 开场15秒检查

**检查项**：
- [ ] 有明确的钩子吗？（反常识/提问/数据/自我暴露）
- [ ] 15秒内说完了吗？
- [ ] 观众会好奇接下来的内容吗？
- [ ] 有没有废话？
- [ ] 能不能再短一点？

**评分标准**：
- 全部满足：⭐⭐⭐⭐⭐（完播率预估：70%+）
- 满足4个：⭐⭐⭐⭐（完播率预估：60-70%）
- 满足3个：⭐⭐⭐（完播率预估：50-60%）
- 满足2个以下：⭐⭐（完播率预估：<50%，需优化）

### 节奏推进检查

**检查项**：
- [ ] 每20-30秒有新信息/转折吗？
- [ ] 有没有连续30秒都在讲同一个点？
- [ ] 故事、案例、金句分布均匀吗？
- [ ] 有推力工具吗？（悬念/转折/提问）

**具体测试**：
```
方法：录音 → 自己听一遍 → 标注无聊点
标准：如果有超过30秒的无聊点，需插入推力工具
```

### 信息密度检查

**检查项**：
- [ ] 每1分钟至少2-3个新观点吗？
- [ ] 有没有重复的表达？
- [ ] 删掉了所有"嗯""啊""那个"吗？
- [ ] 能不能再浓缩一点？

**密度计算**：
```
信息点数量 ÷ 总时长（秒） × 60

目标：2-3个信息点/分钟
实际：X个信息点/分钟

✅ 达标 / ⚠️ 偏低 / ❌ 过载
```

---

## 三、点赞率检查（30%权重）

### 金句质量检查

**检查项**：
- [ ] 数量达标吗？（6-10条）
- [ ] 朗朗上口吗？（能一口气读完）
- [ ] 可截图传播吗？（单独拎出来有传播性）
- [ ] 间隔合适吗？（20-30秒1条）
- [ ] 融入自然吗？（不刻意）

**逐条检查**：
```markdown
金句1："XXX"
- 朗朗上口：✅/❌
- 可截图：✅/❌
- 位置合适：✅/❌
- 融入自然：✅/❌
```

### 共鸣设计检查

**检查项**：
- [ ] 有描述观众困境吗？（"你是不是也..."）
- [ ] 有说出观众内心OS吗？
- [ ] 有给观众解释/解决方案吗？
- [ ] 观众会点头认同吗？

**共鸣层级评估**：
```
L1（表层困境）："每天很忙但不知道忙什么" → 点头
L2（内心冲突）："想躺又焦虑，想卷又累" → 会心一笑
L3（底层恐惧）："30岁还一事无成" → 破防

当前层级：LX
建议层级：L2（既有共鸣又不过度焦虑）
```

### 情绪价值检查

**检查项**：
- [ ] 开场有好奇/共鸣吗？
- [ ] 中段有认同/思考吗？
- [ ] 结尾有受启发/有力量吗？
- [ ] 整体情绪曲线流畅吗？

**情绪曲线**：
```
开场：好奇 ✅/❌
观点1：认同 ✅/❌
观点2：思考 ✅/❌
观点3：顿悟 ✅/❌
结尾：力量 ✅/❌

流畅度：⭐⭐⭐⭐⭐
```

---

## 四、分享率检查（10%权重）

### 实用性检查

**检查项**：
- [ ] 有具体可操作的方法吗？
- [ ] 步骤清晰吗？
- [ ] 普通人能做到吗？
- [ ] 有案例验证吗？
- [ ] 可以立刻开始吗？

**实用性公式验证**：
```
方法 + 具体步骤 + 门槛低 + 可验证

示例对比：
❌ "要提升执行力"
✅ "执行力三步：场景设计 + 交替执行 + 聚焦一件事"

当前实用性：✅足够实用 / ⚠️偏理论 / ❌不够实用
```

### 可传播性检查

**检查项**：
- [ ] 有1-2条15字内的金句吗？
- [ ] 有让人"想转发"的点吗？
- [ ] 有社交货币价值吗？（展示身份/帮助朋友/表达态度）
- [ ] 核心观点可以一句话说清楚吗？

**社交货币评估**：
```
动机1：展示身份（"我也懂这些深度内容"）✅/❌
动机2：帮助朋友（"我朋友需要这个"）✅/❌
动机3：表达态度（"这就是我想说的"）✅/❌
动机4：引发讨论（"你们怎么看？"）✅/❌

满足X个动机 → 分享率预估：XX%
```

---

### 结构完整性检查

**三段式检查**：
- [ ] 开场有钩子吗？（15秒）
- [ ] 中段有3-5个观点吗？（2-3分钟）
- [ ] 结尾有升华吗？（30秒）

**每个观点包含**：
- [ ] 1个反常识观点或核心方法
- [ ] 1-2个故事/案例
- [ ] 1条金句
- [ ] 2-3个疑问句

**结尾必备**：
- [ ] 有"所以"引导总结
- [ ] 重复核心观点
- [ ] 有行动呼吁
- [ ] 最后一句是金句

---

### 故事质量检查

**5要素完整度**：
- [ ] 人物设定清晰吗？（身份+数字）
- [ ] 困境/问题明确吗？（可共鸣）
- [ ] 解决方案新颖吗？（反常识/新方法）
- [ ] 结果具体吗？（有数字）
- [ ] 原理可迁移吗？（观众能用）

**对比效果**：
- [ ] 有强烈对比吗？（A vs B）
- [ ] 对比维度清晰吗？（身份/结果/方法/数字）
- [ ] 对比能制造冲击吗？（让人"哇"）

## 五、常见问题诊断

### 问题1：完播率低

**症状**：
- 平均播放时长 < 60%
- 前15秒流失严重
- 中段明显掉粉

**诊断**：

**原因A：开场不够抓人**
- 解决方案：前15秒必须用四大钩子之一（反常识/提问/数据/自我暴露）
- 检查：15秒内说完了吗？观众会好奇吗？

**原因B：节奏太慢**
- 解决方案：每20-30秒插入新信息/转折
- 检查：有没有连续30秒都在讲同一个点？

**原因C：信息密度不够**
- 解决方案：删掉废话，增加故事/案例/金句
- 检查：能不能删掉1/3的内容？

**原因D：太像讲课**
- 解决方案：增加"你""真的""对吧"等口语化表达
- 检查：像朋友聊天还是像老师讲课？

---

### 问题2：点赞率低

**症状**：
- 完播率正常，但点赞率 < 5%
- 评论区没什么互动
- 收藏/转发也少

**诊断**：

**原因A：没有金句**
- 解决方案：增加6-10条金句，分布在开场/中段/结尾
- 检查：有没有让人想截图的句子？

**原因B：共鸣不够**
- 解决方案：用"你是不是也..."描述观众困境
- 检查：观众会点头吗？

**原因C：情绪价值不足**
- 解决方案：增加自我暴露、失败经历，让观众感受到真实
- 检查：观众会觉得"这人跟我一样"吗？

**原因D：没有启发**
- 解决方案：给出反常识观点或实用方法
- 检查：观众看完会觉得"原来是这样"吗？

---

### 问题3：分享率低

**症状**：
- 完播率、点赞率正常，但分享 < 1%
- 评论区少有"转发给朋友"的留言

**诊断**：

**原因A：不够实用**
- 解决方案：给出具体可操作的方法，降低门槛
- 检查：普通人能立刻开始做吗？

**原因B：金句不够短**
- 解决方案：提炼15字内的金句
- 检查：可以单独截图传播吗？

**原因C：没有社交货币**
- 解决方案：增加有深度的观点或有争议的话题
- 检查：转发这个视频能展示身份/帮助朋友/表达态度吗？

**原因D：核心观点不清晰**
- 解决方案：用"所以"引导总结，重复核心观点
- 检查：观众能一句话说清楚这个视频讲什么吗？

---

### 问题4：内容太刻意

**症状**：
- 评论区有"感觉变了""不如以前真实"的留言
- 老粉丝流失
- 数据正常但感觉不对

**诊断**：

**原因A：金句太密**
- 解决方案：删掉一半金句，只保留最自然的
- 检查：10秒内有连续3条金句吗？

**原因B：技巧太明显**
- 解决方案：回归"朋友深夜聊天感"，减少套路
- 检查：这像在跟朋友聊天吗？

**原因C：失去真诚**
- 解决方案：增加自我暴露、失败经历、真实情绪
- 检查：我自己会转发这条视频吗？

**原因D：为了数据而创作**
- 解决方案：回归初心，想清楚为什么要做这个视频
- 检查：这是我真心想分享的吗？

## 执行流程

**输入**：`broadcast-polish` 的口播稿优化版

逐项检查Q1/Q2/Q3

**如果不通过，停止检查，返回修改**

### Step 3: 爆款三指标检查

按完播率→点赞率→分享率顺序检查

如果指标不达标，根据症状诊断原因
