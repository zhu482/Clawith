# 口播稿-策划

> **从主题到框架，搭建口播稿骨架**

## 零、第一原则

**真诚是底线，"朋友深夜聊天感"是核心。**

所有技巧都要服务于真实感。如果为了指标而变得刻意，立刻停止。

**每次写完必问：**
- [ ] 这像朋友深夜聊天吗？
- [ ] 有没有为了"技巧"而变得刻意？

**如果任何一个是"不"，立刻回退。**

---

## 一、核心认知

### 1.1 爆款三指标（6:3:1策略）

| 指标      | 核心动机               | 关键要素        | 优化重点           |
| ------- | ------------------ | ----------- | -------------- |
| **完播率** | "我想知道结局" + "这对我有用" | 信息密度 + 推力结构 | 开场抓人、悬念递进、故事密度 |
| **点赞率** | "说得太对了" + "我也这么觉得" | 金句 + 反常识    | 共鸣金句、颠覆认知、情绪价值 |
| **分享率** | "我要让朋友看到" + "这个有用" | 可传播性 + 实用性  | 金句截图、方法论、案例启发  |

**原因**：完播率是基础，观众看都看不完，谈何点赞分享？

---

### 1.2 爆款三大支柱

**支柱1：情绪穿透力（60%）**
- 观众40%听内容，60%听情绪
- 让观众觉得你在跟他一对一聊天

**支柱2：信息密度（30%）**
- 短视频不是短的视频，而是浓缩的精华
- 3-7个观点 + 每个观点2-3个故事 + 6-10条金句
- 让观众刷1条 = 刷10条的收获感

**支柱3：故事性（10%）**
- 自古道理留不住，唯有故事得人心
- 观点 → 故事 → 升华
- 让观众能听懂、能记住、能传播

---

### 1.3 三层心理防御破解机制

所有内容之所以能穿透，是因为精准击穿了观众的**三层心理防御**：

| 防御层 | 观众心理 | 破解机制 | 底层原理 |
|--------|----------|----------------|----------|
| **第一层：凭什么听你的？** | 怀疑权威性 | 权威借用（叔叔/书籍/数据） | 认知捷径：借用外部信用 |
| **第二层：这跟我有什么关系？** | 防御性冷漠 | 起手牌理论（我也曾是你） | 镜像神经元：同类人效应 |
| **第三层：看完有什么用？** | 行动瘫痪 | 阶梯推力（你在第几层？） | 损失厌恶：不看完会亏 |

**核心洞察**：
- 普通人在**说服**观众（"你应该相信我"）
- 高手在**设计**观众的认知路径（"你自己会得出结论"）

**所以，真诚不是软弱，而是最强大的破防武器。**

---

### 1.4 三个常见误区

❌ **误区1：讲得越多越好** → 信息过载会让观众逃避，真正让人看下去的是**节奏**，不是数量  
❌ **误区2：金句越多越好** → 核心是**恰到好处**，金句太多会显得刻意，关键位置放金句  
❌ **误区3：反常识 = 抬杠** → 要说出观众想说但说不出来的话

## 二、开场钩子设计

### 开场黄金15秒

**核心原则**：15秒内必须给观众一个"看下去的理由"

**四大钩子类型**：

**钩子1：反常识抛出**（最爆）
> "你以为聪明人赢在才华，其实他们赢在愿意花笨功夫"

**钩子2：提问引发思考**（最稳）
> "如果你每天只能做一件事，那这件事情是什么？"

**钩子3：数据震撼**（最有力）
> "同样拍视频，为什么流浪大师百万点赞，西装男士只有100赞？"

**钩子4：自我暴露**（最真实）
> "我作废了六个账号，才搞明白这件事..."

**检查清单**：
- [ ] 15秒内说完了吗？
- [ ] 观众会好奇吗？
- [ ] 有没有废话？
- [ ] 能不能再短一点？

---

### 深层原理：预期违背机制

所有情绪来自"预期"与"现实"的落差。钩子设计 = **先建立预期** + **然后打破预期**

**类型A：身份预期违背**
```
【建立预期】：流浪汉 = 没文化
【打破预期】：流浪汉念京剧 → 300万点赞

【建立预期】：北大毕业 = 天才
【打破预期】："你应该强调你18岁前没坐过火车"
```

**类型B：观点预期违背**
```
【建立预期】：努力 = 成功
【打破预期】："财富的源代码从来不是努力"
```

**核心**：不是"标题党"，而是**真的有反常识洞察**。

---

### 钩子增效：权威借用

**原理**：观众的第一反应是"凭什么听你的？"

你说"我觉得X很重要" → 观众：谁在乎你怎么想？  
你说"这是某某教我的X" → 观众：那我听听看

**三种权威来源**：

**来源1：经验权威**（最可信）
```
✅ "这是我从商30年的叔叔教我的"
✅ "这是我做了8年自媒体总结出来的"
✅ "这是我作废6个账号后发现的"
```
→ 关键：有具体数字 + 有代价成本

**来源2：知识权威**（最专业）
```
✅ "《纳瓦尔宝典》里有句话..."
✅ "斯坦福大学研究发现..."
✅ "某上市公司创始人说过..."
```
→ 关键：可追溯 + 有知名度

**来源3：数据权威**（最有力）
```
✅ "我统计了100个爆款视频，发现..."
✅ "对比了500个账号后..."
```
→ 关键：样本量够大 + 有对比

**组合公式**：
```
[权威来源] + [反常识观点] + [痛点承诺]

示例：
"从小到大，从来没有人教过我们怎么赚钱。
但我叔叔从商30年，告诉我一个颠覆认知的道理..."
```

**自检**：
- [ ] 有没有"我觉得"、"我认为"？→ 换成权威来源
- [ ] 权威来源可信吗？→ 加数字/书名/身份
- [ ] 能追溯验证吗？→ 给出具体出处

## 三、结构模板选择

### 3.1 万能三段式思路

**适用**：80%的主题

**结构思路**：
```
开场钩子（15秒）→ 三个观点递进（2-3分钟）→ 结尾升华（30秒）
```

**开场钩子（选一种思路）**：

**钩子A：反常识抛出**
> "你以为聪明人赢在才华，其实他们赢在愿意花笨功夫"

**钩子B：提问引发思考**
> "你现在想一个问题，如果你每天只能做一件事，那这件事情是什么？"

**钩子C：自我暴露**
> "我作废了六个账号，终于搞明白一件事..."

**选择标准**：
- 有反常识观点 → 选A
- 主题偏深度思考 → 选B  
- 有失败经历 → 选C

---

#### 三观点递进逻辑

**观点数量选择**：

| 观点数 | 优点 | 缺点 | 适用场景 |
|--------|------|------|---------| 
| 1-2个 | 聚焦、深入 | 信息密度不够 | 单一方法教学 |
| 3个 | 完整、好记 | 标准配置 | 万能模板 ⭐ |
| 5个 | 丰富、干货 | 需要强大叙事 | 系统方法论 |
| 7个 | 信息爆炸 | 难以记忆 | 超长干货 |
| 10+个 | 量大管饱 | 容易走神 | 避免使用 |

**最佳配置：3观点**
- 开始-过程-结束的完整性
- 认知负荷刚好
- 三段式朗朗上口

**递进逻辑**：

**观点1**：是什么（定义问题）
- 用案例说明现象
- 抛出反常识观点

**观点2**：为什么（分析原因）
- 讲故事证明
- 给出底层逻辑

**观点3**：怎么办（解决方案）
- 提供具体方法
- 用案例验证

**每个观点标配**：
```
1个反常识观点
+ 1-2个故事/案例
+ 1条金句
+ 2-3个疑问句
```

**增效机制：阶梯推力**

如果你的主题可以分层级、分段位，用这个方法能让完播率飙升。

**原理**：人脑讨厌"未完成的认知"，阶梯式递进会让人停不下来。

**对比示例**：

❌ **平铺式（观众看完第1个就走了）**：
```
赚钱有3个方法：
1. 做自媒体
2. 做电商
3. 投资理财

每个都试试看吧。
```
→ 问题：看完第1个就知道后面是啥了，认知缺口已关闭。

✅ **阶梯式（观众会一直看到最后）**：
```
赚钱有3个段位：

Level 1：卖时间（90%的人困在这里，月入5k-1w）
→ 你现在在这一层吗？

Level 2：卖产品（少数人突破了，月入1w-10w）
→ 但这还不是终点...

Level 3：买时间（只有1%的人知道，月入10w+）
→ 这才是真正的财富密码
```

**为什么有推力？**
1. **身份对号**："我在第几层？"（焦虑+好奇）
2. **进阶暗示**："后面更牛逼"（期待感）
3. **稀缺暗示**："只有1%知道"（FOMO恐惧）

**万能公式**：
```
[话题] 的 N 个段位/层级/阶段：

Level 1：[大多数人的现状] + 特征描述
→ 制造焦虑：你现在在这一层吗？

Level 2：[少数人的突破] + 成果展示
→ 制造向往：但这还不是终点...

Level 3：[极少数人的秘密] + 稀缺性
→ 制造紧迫：这才是真正的X
```

**适用主题**：
- 赚钱方法 → 卖时间/卖产品/买时间
- 能力提升 → 知道/做到/教到
- 认知升级 → 表象/本质/规律

**关键**：不是"并列关系"，而是"递进关系"。

**自检**：
- [ ] 观众看完第1段会想看第2段吗？
- [ ] 有没有"后面更牛逼"的暗示？
- [ ] 每个段位结尾有没有悬念？

---

#### 结尾升华

**公式**：
```
所以 + 核心观点 + 行动呼吁
```

**经典模板**：
> "所以你要记住，真正的优势不是别人看得见的，而是你自己看得见的。从今天开始，找到你的优势，然后..."

**必备要素**：
- [ ] 有"所以"引导总结
- [ ] 重复核心观点
- [ ] 给出行动方向
- [ ] 最后一句是金句

---

### 3.2 悬念式（最抓人）

**适用场景**：有强对比案例、有反转故事

**结构**：
```
抛出悬念（15秒）
  ↓
案例对比/故事展开（2分钟）
  ↓
揭晓答案/原理升华（1分钟）
```

---

#### 悬念设置公式

**公式1：反差悬念**
> "同样是拍视频，为什么流浪大师百万点赞，西装男士只有100赞？"

**公式2：结果悬念**
> "我花了3年时间，作废了6个账号，才搞明白这个道理..."

**公式3：方法悬念**
> "北大毕业年薪百万，为什么卖199的课程都卖不出去？"

**检查标准**：
- [ ] 有明确对比吗？
- [ ] 观众会好奇答案吗？
- [ ] 15秒内能说完吗？

---

#### 案例展开策略

**展开公式**：
```
A做法 → A结果（失败/普通）
B做法 → B结果（成功/爆发）
核心差异是什么？
```

**案例细节密度**：
- 身份标签（北大、34岁、年薪百万）
- 具体数字（199元课程、100赞vs百万赞）
- 关键对话（还原现场感）
- 情绪描述（让观众代入）

**时间分配**：
- A案例：40秒
- B案例：60秒
- 对比分析：20秒

---

#### 揭晓答案要点

**核心**：不是简单给答案，而是层层递进

**递进公式**：
```
表面原因（大家都能想到）
  ↓
中层原因（需要思考）
  ↓
底层逻辑（反常识认知）
```

**经典案例**：

**表面**：流浪大师火是因为反差  
**中层**：观众喜欢真实感  
**底层**：真实就是最大的稀缺

**升华关键**：
- [ ] 有抽象原理吗？
- [ ] 可迁移到其他场景吗？
- [ ] 观众能记住吗？

---

### 3.3 方法论式（最干货）

**适用场景**：教方法、讲技巧、拆解系统

**结构**：
```
痛点共鸣（30秒）
  ↓
方法拆解（3-5个步骤，2-3分钟）
  ↓
效果验证（30秒）
```

---

#### 痛点共鸣公式

**公式**：
```
你是不是也... （描述困境）
很多人都... （建立同理心）
其实问题出在... （引出解决方案）
```

**经典案例**：
> "你每天都很忙，但却不知道在忙什么，大钱赚不到，小钱看不上，最后什么都没得到。其实问题出在..."

**检查标准**：
- [ ] 观众会点头吗？
- [ ] 有具体场景吗？
- [ ] 引出方法自然吗？

---

#### 方法拆解要点

**方法数量**：3-5个（最佳3个）

**每个方法包含**：
```
1. 方法名称（可记忆）
2. 核心原理（为什么有效）
3. 具体步骤（怎么做）
4. 案例验证（谁用过）
```

**经典案例**：

**方法1：目标拆解**
- 原理：大目标会瘫痪，小目标会行动
- 步骤：月目标→周目标→日任务
- 案例：我从月薪15000到月入10万+就是这样拆的

**方法2：即刻行动**
- 原理：开始永远大于完美
- 步骤：5秒法则，想到立刻做
- 案例：我19岁做抖音，第一条就发了

**方法3：持续迭代**
- 原理：进步来自反馈，不是计划
- 步骤：做→收集反馈→调整→再做
- 案例：作废6个账号才摸索出套路

---

#### 效果验证策略

**验证公式**：
```
我用这个方法 + 具体数字成果 + 你也可以
```

**经典案例**：
> "我就是用这三步，从月薪15000的程序员到现在月入10万+。所以你也可以，从今天开始..."

**必备要素**：
- [ ] 有个人经历
- [ ] 有具体数字
- [ ] 有行动呼吁
- [ ] 给观众信心

## 执行流程

### Step 1: 接收主题

**用户提供**：
- 主题/选题
- 素材（可选）
- 目标人群（可选）

### Step 2: 定锚分析

询问用户（如果信息不足）：
1. 这个主题主要给谁看？（目标人群）
2. 他们的核心痛点是什么？
3. 你有什么独特经历/观点/案例？

输出：
```markdown
## 定锚分析
- 目标人群：XXX
- 核心痛点：XXX
- 破防策略：
  - 第一层（权威）：XXX
  - 第二层（关联）：XXX
  - 第三层（推力）：XXX
```

### Step 3: 选择钩子类型

根据主题特点，推荐1-2个钩子：
- 有反常识 → 钩子1
- 偏思考 → 钩子2
- 有数据对比 → 钩子3
- 有失败经历 → 钩子4

### Step 4: 选择结构模板

| 主题类型 | 推荐模板 |
|---------|---------|
| 能力提升 | 三段式/方法论式 |
| 赚钱创业 | 悬念式 |
| 认知升级 | 三段式 |

### Step 5: 搭建观点框架

根据选定模板，设计3-7个观点的递进逻辑

### Step 6: 输出策划文档

```markdown
# 【口播稿策划文档】

## 一、定锚分析
（内容）

## 二、开场钩子（15秒内）
【方案A】XXX
【方案B】XXX
👉 推荐：XXX

## 三、结构模板
选择：XXX
理由：XXX

## 四、观点框架
观点1：XXX
观点2：XXX
观点3：XXX

## 五、预期效果
完播率预估：XX%
点赞率预估：XX%
核心记忆点：XXX
```
