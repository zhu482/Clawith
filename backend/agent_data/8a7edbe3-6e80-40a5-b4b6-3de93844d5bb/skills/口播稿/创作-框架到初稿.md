# 口播稿-创作

> **从框架到内容，写出口播稿初稿**

## 零、第一原则

**真诚是底线，"朋友深夜聊天感"是核心。**

每次写完必问：
- [ ] 这像朋友深夜聊天吗？
- [ ] 有没有为了"技巧"而变得刻意？

**如果任何一个是"不"，立刻回退。**

## 一、情绪穿透力设计

**核心原理**：小屏幕削弱情绪，必须用表达力抵消

### 方法1：口语化表达

**疑问引导**："你知道吗？""对吧？""你发现了吗？""为什么？"

**强化副词**："真的""非常""特别""竟然""太...了""从来"

**转折词**："但是""所以""那么""其实"

**情绪词**："我觉得""我发现""我想说"

**节奏公式**：
```
短句 + 疑问 + 强调 + 举例
↓
"为什么？很简单，因为..."
"你发现了吗？其实..."
"所以我就想..."
```

### 方法2：第二人称叙事

- 全程用"你"而非"大家"
- ✅ "你一定要记住..."（代入感、聊天感）

**典型话术**：
> "你现在想一个问题，如果你每天只能做一件事，那这件事情是什么？"

### 方法3：自我暴露

- 失败经历："我作废了六个账号"、"我以前就是这样，眼高手低，大钱赚不到，小钱看不上"
- 成长过程："从月薪15000的程序员到裸辞创业"
- 底层逻辑：我的起手牌 ≤ 你的起手牌 → 可信度↑
- 经典案例："你应该强调的是你18岁前没坐过火车，好不容得熬到今天才年薪百万，这才应该是你的人生"
- 本质：人看到比自己差的人都成功才会相信

## 二、信息密度设计

**核心原理**：让观众刷1条 = 刷10条的收获感

**但密度不是"知识密度"，而是"认知爽点密度"**

### 什么是认知爽点？

不是"塞很多知识点"，而是让观众产生三种感觉：

**爽点1：顿悟感**  
→ "哦，原来是这样！"  
→ 来源：反常识观点、新颖比喻、原理揭秘

**爽点2：共鸣感**  
→ "我靠，说得太对了！"  
→ 来源：痛点描述、内心OS、失败经历

**爽点3：获得感**  
→ "这个我能用！"  
→ 来源：具体方法、可操作步骤、公式模型

### 密度公式

**目标**：每30秒1个认知爽点

**对比示例**：

❌ **低密度（纯理论，观众走神）**：
```
【0-30秒】：解释什么是"底层能力"
【30-60秒】：解释为什么重要
【60-90秒】：解释怎么培养
```
→ 问题：连续讲道理90秒，观众大脑拒绝接收

✅ **高密度（理论+故事+金句，观众停不下来）**：
```
【0-30秒】：观点 + 金句
"执行力是离钱最近的能力"
→ 爽点：顿悟感

【30-60秒】：故事1（自我暴露）
"我以前眼高手低，大钱赚不到，小钱看不上"
→ 爽点：共鸣感

【60-90秒】：故事2（具体方法）
"后来我学会了场域设计，在图书馆就想学习，
因为没有游戏的能量场"
→ 爽点：获得感

【90-120秒】：金句总结
"如果执行力跟不上认知，那就是精神内耗"
→ 爽点：顿悟感
```

**为什么有密度？**
- 每30秒都有"新信息"（知识爽点）
- 每30秒都有"新故事"（情绪爽点）
- 每30秒都有"新金句"（传播爽点）

### 密度配方

**1个视频配方**：
```
3-7个核心观点
+ 每个观点2-3个故事/案例
+ 6-10条金句
+ 15-25个疑问句
+ 若干数字锚点
```

**信息间距控制**：
- 目标：每20-30秒1个信息点
- 信息点类型：
  1. **观点抛出**（最重要）
  2. **案例故事**（最稳）
  3. **金句总结**（最爽）
  4. **数字数据**（最有力）
  5. **悬念转折**（最抓人）

**密度红线**：
- ❌ 连续1分钟讲道理
- ❌ 连续30秒没新信息
- ❌ 让观众"知道你接下来要说什么"

**自检公式**：
```
信息点数量 ÷ 总时长（秒） × 60 = X个信息点/分钟

目标：2-3个信息点/分钟
✅ 达标 / ⚠️ 偏低（增加故事/金句） / ❌ 过载（删减重复）
```

## 三、故事性设计

**核心理念**：自古道理留不住，唯有故事得人心

**故事类型分布**（基于爆款分析）：
- 个人经历：40%（转行、失败、成长）
- 身边案例：30%（北大表哥、舅舅炒饭、清华博士收废品）
- 名人案例：20%（乔丹、马云、董宇辉）
- 隐喻类比：10%（电影院vs手机、松鼠大象过河）

### 抽象概念具象化

**原理**：大脑不擅长处理抽象概念，但擅长记忆具体画面。

**对比示例**：

❌ **抽象表达（听不懂）**：
```
"你要学会透过现象看本质"
```
→ 问题：什么是"本质"？怎么"透过"？脑海里没有画面。

✅ **具象表达（秒懂）**：
```
"松鼠过河步步惊心，大象过河漫不经心。
同样的河，为什么感受不同？
因为松鼠看到的是'危险'，大象看到的是'小水坑'。
这就是认知差距。"
```
→ 为什么有效？
- 大脑自动脑补画面（松鼠vs大象）
- 对比产生认知冲击（步步惊心vs漫不经心）
- 隐喻可迁移（我是松鼠还是大象？）

**具象化公式**：
```
步骤1：识别抽象词（本质、认知、思维、能力）
步骤2：找对应的具象物（动物、场景、人物、物品）
步骤3：用对比制造冲击（A vs B）

模板：
- "[抽象概念] 就像 [具象比喻]"
- "[A做法] 像 [具象物A]，[B做法] 像 [具象物B]"
```

**常用抽象词替换表**：

| 抽象词 | 具象化比喻 | 效果 |
|--------|-----------|------|
| 认知差距 | 松鼠vs大象过河 | 有画面、有对比 |
| 精神内耗 | 认知在天上飞、行动在地上爬 | 形象、易记 |
| 注意力分散 | 信息的肥胖者 | 有冲击力 |
| 场景影响行为 | 电影院vs手机看电影 | 可共鸣 |
| 优势+趋势 | 舅舅卖炒饭（食材优势） | 接地气 |

**自检**：
- [ ] 有没有"本质"、"认知"、"思维"这些抽象词？
- [ ] 能不能用一个"有画面感"的比喻替代？
- [ ] 观众能不能在脑海里想象出画面？

### 罗伯特故事结构

**三段式**：
1. **抛观点** → 引发好奇
2. **讲故事** → 具象化理解
3. **归纳法则** → 升华认知

**经典案例**：

**观点**：优势+趋势=赚钱（老掉牙）

**故事1**：舅舅卖炒饭形成食材优势，月入3万+  
**故事2**：清华博士收废品，降维打击年入百万

**升华**：形成优势，顺应趋势

**效果**：老掉牙观点讲出1000万播放量

---

### 顶级故事设计公式

**三明治结构**（道理+故事+总结）

**原理**：道理太干，故事太水，三明治刚刚好。

**结构公式**：
```
【上层面包】：抛出观点/结论
【中间夹心】：讲一个具体故事佐证
【下层面包】：总结升华，强化记忆
```

**示例**：
```
【上层】："老实人赚不到大钱"
→ 制造冲击：什么？为什么？

【夹心】："我有个朋友是北大毕业的，很老实，
        开诚布公地告诉客户自己产品的不足，
        结果客户转头就找别人了。
        另一个学历不高的人，懂得包装亮点、
        先建立信任，反而签下大单。"
→ 具象化理解：哦，原来是这样

【下层】："所以，不要做老实人，要做聪明的好人"
→ 强化记忆：我记住了
```

**为什么有效？**
- 上层面包：制造认知冲击（"什么？"）
- 中间夹心：具象化理解（"哦，原来如此"）
- 下层面包：强化记忆（"我记住了"）

**自检**：
- [ ] 有没有连续3段都在讲道理？→ 插入故事
- [ ] 有没有讲了故事但没总结？→ 加上下层面包
- [ ] 故事太长观众忘了观点？→ 缩短故事或中间提醒

### 故事的信任机制：起手牌理论

**原理**：人不相信"道理"，但相信"经历"。更关键的是，**人只会买同类人的产品**。

**对比示例**：

❌ **天才叙事（观众不信）**：
```
"我北大毕业，34岁年薪百万，
所以我的方法一定有用。"
```
→ 观众心理："你是天才，我学不会"

✅ **同类人叙事（观众认同）**：
```
"我18岁前没坐过火车，
一步步摸爬滚打，
才熬到今天年薪百万。"
```
→ 观众心理："他比我还惨，都能成功，我也能"

**起手牌公式**：
```
起手牌（我也曾是你）+ 转折（但我做了X）+ 结果（所以你也能）

判断标准：
我的起手牌 ≤ 观众的起手牌 → 可信度↑
我的起手牌 > 观众的起手牌 → 可信度↓
```

**案例类型**：

**个人经历型**：
- ✅ "我作废了6个账号"（强调代价）
- ✅ "我以前眼高手低，大钱赚不到，小钱看不上"（强调失败）
- ✅ "我从月薪15000的程序员"（强调起点低）

**身边案例型**：
- ✅ "我有个朋友，只有高中学历..."
- ✅ "我舅舅卖炒饭，每天起早贪黑..."
- ✅ "我见过一个宝妈，带娃的同时..."

**自检**：
- [ ] 我的起手牌是否 ≤ 观众的起手牌？
- [ ] 有没有强调"成本/代价/失败"？
- [ ] 观众会觉得"这人跟我一样"吗？

---

### 顶级故事设计公式（5要素）

**5要素完整度**：

1. **人物设定**（身份+数字）
   - 北大毕业、34岁、年薪百万
   
2. **困境/问题**（可共鸣）
   - 卖199课程卖不出去

3. **解决方案**（反常识/新方法）
   - 不强调北大，强调18岁前没坐过火车

4. **结果**（具体数字）
   - （未明说但暗示会成功）

5. **原理升华**（可迁移）
   - 人只会买同类人的产品

**为什么这个案例强？**
- ✅ 反常识（北大高材生失败）
- ✅ 有具体数字（34岁、年薪百万、199元）
- ✅ 有对话（真实感）
- ✅ 有原理升华
- ✅ 可迁移（所有人都能用）

---

### 对比制造认知冲击

**核心**：所有案例都有强对比

**对比类型**：

| 对比维度 | 案例 | 效果 |
|---------|------|------|
| 身份反差 | 流浪大师vs西装男士 | 百万赞vs100赞 |
| 结果反差 | 北大表哥失败vs普通人成功 | 认知颠覆 |
| 方法对比 | 普通人拼才华vs聪明人拼执行力 | 实用性凸显 |
| 数字对比 | 15000程序员vs月入10万+ | 动机强化 |

**使用公式**：
```
A类人 + 常规做法 = 失败/普通
B类人 + 反常识做法 = 成功/爆发
↓
核心：做法是关键，不是天赋
```

## 四、金句系统

**核心理念**：金句不是文字游戏，而是观众想说但说不出的话

**密度配置**：6-10条/视频
- 开场钩子：1条（15秒内）
- 中间节点：4-6条（每1-2分钟1条）
- 结尾升华：2-3条（最后30秒）
- **密度红线**：❌ 10秒内连续3条 / ✅ 20-30秒间隔1条

### 金句四大类型

**A. 反常识型**（最爆）

核心：颠覆固有认知

**金句公式**：对比 + 对称 + 绝对化

**对比示例**：

❌ **平庸表达（没记忆点）**：
```
"有钱人和穷人的思维方式不一样"
```
→ 问题：没有冲击力，说了等于没说

✅ **金句表达（可传播）**：
```
"有钱人敢于在社交里吃亏，
而穷人一生都在追求绝对的公平"
```
→ 为什么有效？
- **对比**：有钱人 vs 穷人（制造冲击）
- **对称**：吃亏 vs 公平（易于记忆）
- **绝对化**："一生"、"绝对"（强化观点）

**万能公式**：
```
[A群体][做法A]，而[B群体][做法B]

模板：
- "普通人[X]，而牛人[Y]"
- "不是[常识]，而是[真相]"
- "[X]从来不是[Y]，而是[Z]"

关键：对比必须有"认知冲击"
```

**经典案例**：
> "大家都想要高情商，其实情商分善意和恶意，恶意情商就是PUA"
> "你觉得聪明人赢在才华，其实他们赢在愿意花笨功夫"
> "你以为时间是自由的，其实自由本身就是稀缺的"

**爆款指数**：⭐⭐⭐⭐⭐

---

**B. 升华型**（最有深度）

核心：从具体案例提炼普世法则

**公式**：
```
所以 + 抽象原理
这就是为什么 + 底层逻辑
```

**经典案例**：
> "所以真正的优势，不是别人看得见的，而是你自己看得见的"
> "这就是为什么会延误，因为你本以为他会按时赴约，结果他竟然迟到了"
> "人只会买同类人的产品"

**爆款指数**：⭐⭐⭐⭐

---

**C. 共鸣型**（最稳）

核心：说出观众想说但说不出的话

**公式**：
```
你是不是也... 
很多人都... 其实...
```

**经典案例**：
> "很多人大钱赚不到，小钱看不上，最后什么都没得到"
> "你每天都很忙，但却不知道在忙什么"
> "你觉得自己在努力，其实只是在重复劳动"

**爆款指数**：⭐⭐⭐⭐

---

**D. 行动型**（最实用）

核心：可落地的方法论

**公式**：
```
记住这X点...
下次遇到...时，你就...
```

**经典案例**：
> "每天只做一件事，但这件事一定要有复利"
> "你要训练一个能力，就是在认知和现实之间找到平衡点"
> "执行力 = 目标拆解 × 即刻行动 × 持续迭代"

**爆款指数**：⭐⭐⭐

---

### 金句位置策略

**关键位置必放金句**：

1. **开场15秒**（决定完播率）
   - 用反常识金句抓眼球
   - ✅ "你以为..."型
   
2. **每1分钟节点**（维持注意力）
   - 用共鸣型金句强化认同
   - ✅ "你是不是也..."型
   
3. **结尾30秒**（决定点赞率）
   - 用升华型金句制造余味
   - ✅ "所以真正的..."型

---

### 金句检查清单

写完金句后，逐条检查：

- [ ] **朗朗上口**：能一口气读完吗？
- [ ] **可截图**：单独拎出来有传播性吗？
- [ ] **有画面感**：能让人脑海里出现画面吗？
- [ ] **不刻意**：融入上下文自然吗？
- [ ] **有情绪**：能让人点头/会心一笑吗？

**如果任何一条不满足，重写。**

## 五、话术模板库

### 模板1：能力提升类

**适用主题**：执行力、学习力、决策力、自律等

#### 结构框架

```
开场（15秒）：反常识观点
  ↓
痛点共鸣（30秒）：你是不是也...
  ↓
三个方法（2分钟）：具体可操作
  ↓
效果验证（30秒）：我的经历
  ↓
结尾升华（30秒）：行动呼吁
```

#### 话术模板

**【开场钩子】**
> "你以为[能力名称]靠的是[常见认知]，其实真正有[能力]的人，靠的是[反常识观点]。"

**示例**：
> "你以为执行力靠的是自律，其实真正执行力强的人，靠的是行为设计。"

---

**【痛点共鸣】**
> "你是不是也这样，[描述困境1]，[描述困境2]，最后[糟糕结果]。其实问题不是你不够努力，而是[根本原因]。"

**示例**：
> "你是不是也这样，认知很高但就是做不到，看了很多道理但还是过不好这一生，最后陷入精神内耗。其实问题不是你不够努力，而是你的执行力跟不上你的认知。"

---

**【方法1：场景设计】**
> "第一个方法，[方法名称]。举个例子，[具体案例]。你发现了吗？[原理解释]。"

**示例**：
> "第一个方法，场景设计。举个例子，你在电影院看电影和在手机上看电影，完全是两个体验对吧？因为场景会限制你的行为。所以你要给自己设计专门的工作场景，在书房就不要打游戏，想休息就去客厅。"

---

**【方法2：具体行为】**
> "第二个方法，[方法名称]。我自己就是这样做的，[个人经历]。后来我发现，[效果/原理]。"

**示例**：
> "第二个方法，交替执行。我自己会准备多个工作场景，书房待烦了就去星巴克，星巴克待烦了就去书店。这样你永远有新鲜感，执行力自然就上来了。"

---

**【方法3：心法升华】**
> "第三个方法，也是最重要的，[方法名称]。[名人/权威]说过，[金句]。所以[具体应用]。"

**示例**：
> "第三个方法,也是最重要的，聚焦一件事。查理·芒格说过，最重要的事只有一件。所以每天早上问自己，今天最重要的事是什么？先把它完成，其他的都不重要。"

---

**【效果验证】**
> "我就是用这三个方法，从[起点]到[终点]。所以[鼓励观众]，你也可以做到。"

**示例**：
> "我就是用这三个方法，从月薪15000的程序员到现在月入10万+。所以别焦虑，行动起来，你也可以做到。"

---

**【结尾升华】**
> "所以你要记住，[核心观点]。真正的[能力名称]不是[错误认知]，而是[正确认知]。从今天开始，[行动呼吁]。"

**示例**：
> "所以你要记住，认知是起点，执行才是终点。真正的高手不是知道得多，而是做到得多。从今天开始，找到你最重要的那件事，然后立刻去做。"

### 模板2：赚钱/创业类

**适用主题**：副业、创业、赚钱思维等

#### 结构框架

```
开场（15秒）：数字震撼/身份反差
  ↓
观点抛出（30秒）：[公式/原理]
  ↓
案例1（1分钟）：失败案例
  ↓
案例2（1分钟）：成功案例
  ↓
原理升华（30秒）：可迁移方法论
  ↓
行动建议（30秒）：具体第一步
```

#### 话术模板

**【开场钩子】**
> "同样是[做某事]，为什么[A类人]能[好结果]，而[B类人]只能[差结果]？"

**示例**：
> "同样是做自媒体，为什么流浪大师能百万点赞，而西装革履的职场博主只有100赞？"

---

**【观点抛出】**
> "答案很简单，因为[核心公式/原理]。"

**示例**：
> "答案很简单，因为真实就是最大的稀缺。观众不是不喜欢内容，而是不喜欢装。"

---

**【失败案例】**
> "[人物身份]，[初始条件很好]，但他[常规做法]，结果[失败/普通]。为什么？因为[错误认知]。"

**示例**：
> "我有个表哥，北大毕业，34岁年薪百万，卖199元的课程。但他在视频里一直强调自己北大毕业、年薪百万，结果根本卖不出去。为什么？因为人只会买同类人的产品。"

---

**【成功案例】**
> "但如果他这样做呢？[反常识做法]。[描述具体细节]。你发现了吗？[原理解释]。"

**示例**：
> "但如果他不强调北大，而是强调自己18岁前连火车都没坐过，好不容易熬到今天才年薪百万呢？观众一听，哇，这人跟我一样苦过，他的方法我也能用。这才是正确的打开方式。"

---

**【原理升华】**
> "所以真正的[关键词]不是[错误认知]，而是[正确认知]。这就是[核心原理]。"

**示例**：
> "所以真正的优势不是北大学历、不是年薪百万，而是和用户建立同理心。这就是为什么草根逆袭的故事永远比精英成功更有市场。"

---

**【行动建议】**
> "如果你也想[目标]，记住这三点：第一，[具体动作1]。第二，[具体动作2]。第三，[具体动作3]。"

**示例**：
> "如果你也想通过自媒体赚钱，记住这三点：第一，别装，展示真实的自己。第二，讲你的苦难和成长，不要炫耀。第三，用结果说话，不要画大饼。"

### 模板3：认知升级类

**适用主题**：思维方式、底层逻辑、元能力等

#### 结构框架

```
开场（15秒）：提问引发思考
  ↓
观点1（1分钟）：定义/现象
  ↓
观点2（1分钟）：原因/逻辑
  ↓
观点3（1分钟）：方法/升华
  ↓
结尾（30秒）：行动+金句
```

#### 话术模板

**【开场提问】**
> "你现在想一个问题，[引发深度思考的问题]。这个问题的答案，决定了[重要结果]。"

**示例**：
> "你现在想一个问题，如果你每天只能做一件事，那这件事情是什么？这个问题的答案，决定了你未来三年的高度。"

---

**【观点1：定义】**
> "很多人觉得[能力/概念]就是[常见认知]，其实不是。真正的[能力/概念]是[新定义]。"

**示例**：
> "很多人觉得执行力就是自律、就是坚持，其实不是。真正的执行力是认知和行动的匹配度。"

---

**【观点2：原因】**
> "为什么这么说？因为[案例/故事]。你发现了吗？[底层逻辑]。"

**示例**：
> "为什么这么说？因为我见过太多人，认知在天上飞，行动在地上爬，最后陷入精神内耗。你发现了吗？真正让人痛苦的不是做不到，而是知道但做不到。"

---

**【观点3：方法】**
> "所以你要[核心方法]。具体怎么做？[步骤1 + 步骤2 + 步骤3]。"

**示例**：
> "所以你要训练一个能力，就是在认知和现实之间找到平衡点。具体怎么做？第一，降低认知（专注一件事）。第二，提高执行（场景设计）。第三，持续迭代（小步快跑）。"

---

**【结尾升华】**
> "记住，[金句总结]。[名人/权威]说过，[名言]。从今天开始，[行动呼吁]。"

**示例**：
> "记住，知行合一不是知道然后做到，而是在做的过程中不断调整认知。王阳明说，知行合一。从今天开始，少想多做，你会发现一切都会不一样。"

## 执行流程

### Step 1: 接收策划文档

**输入**：`broadcast-strategy` 的策划文档

### Step 2: 调用案例库

**必须调用** `broadcast-case-library` 获取：
- 相关案例
- 金句参考
- 话术模板

### Step 3: 选择话术模板

根据主题类型选择：
- 能力提升 → 模板1
- 赚钱创业 → 模板2
- 认知升级 → 模板3

### Step 4: 填充内容

按模板填充：
- 开场钩子
- 观点+故事+金句
- 结尾升华

### Step 5: 检查密度

- 金句数量：6-10条 ✓
- 故事数量：3-6个 ✓
- 信息点：每20-30秒1个 ✓

### Step 6: 输出初稿

```markdown
# 【口播稿初稿】主题：XXX

## 【开场】（15秒）
XXX

## 【观点1】（1分钟）
XXX

## 【观点2】（1分钟）
XXX

## 【观点3】（1分钟）
XXX

## 【结尾】（30秒）
XXX

---

## 【创作说明】
- 总时长：X分X秒
- 金句数量：X条
- 故事数量：X个
- 信息点密度：每XX秒1个
```

## 检查清单

- [ ] 像朋友聊天吗？（一票否决）
- [ ] 每个观点都有完整的故事吗？
- [ ] 金句数量达标吗？（6-10条）
- [ ] 每20-30秒有新信息吗？
- [ ] 用"你"而不是"大家"吗？

## 禁忌事项

❌ **不要**：
- 连续1分钟讲道理
- 连续30秒没新信息
- 金句太密（10秒内连续3条）
- 用书面语
- 让观众"知道你接下来要说什么"
