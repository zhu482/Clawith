# 检索 - 统一信息检索工具

统一的信息检索入口，支持本地检索、向量库检索和网络搜索三种方式。

## 功能说明

统一的信息检索入口，支持本地检索、向量库检索和网络搜索三种方式。

## 使用方法

### 1. 智能检索（推荐）
```
/检索 [关键词]
```
自动选择最佳检索方式，优先本地检索，必要时调用向量库和网络搜索。

### 2. 本地检索
```
/检索 本地 [关键词]
```
仅在本地文件中搜索，包括创作空间、高质样例、个人信息等。

### 3. 向量库检索
```
/检索 向量库 [关键词]
```
使用向量库在知识库中检索相关信息。

### 4. 网络搜索
```
/检索 网络 [关键词]
```
使用网络搜索引擎查找最新信息。

## 示例

```bash
# 智能检索
/检索 AI写作方法

# 本地检索
/检索 本地 公众号运营
/检索 本地 品味

# 向量库检索
/检索 向量库 深度学习
/检索 向量库 产品设计

# 网络搜索
/检索 网络 Claude 3.5 Sonnet 最新功能
/检索 网络 2026年AI趋势
```

## 执行流程

### 智能检索模式

1. **本地优先检索**
   - 使用Grep工具在本地文件中搜索关键词
   - 检索范围: 创作空间、高质样例、个人信息
   - 如果找到5条以上相关结果，直接返回

2. **向量库补充**
   - 如果本地结果不足，调用向量库检索
   - 补充更多相关信息

3. **网络搜索兜底**
   - 如果仍需要更多信息，进行网络搜索
   - 获取最新的互联网信息

4. **结果汇总**
   - 整合三种来源的信息
   - 按相关度排序
   - 提供信息来源标注

### 本地检索模式

1. **确定检索范围**
   - `创作空间/` - 用户的创作内容
   - `高质样例/` - 公众号文章知识库
   - 个人信息及记忆文件

2. **执行搜索**
   - 使用Grep工具搜索关键词
   - 支持中文和英文
   - 支持多个关键词（空格分隔）

3. **显示结果**
   - 匹配的文件路径
   - 匹配的上下文片段
   - 按相关度排序

### 向量库检索模式

1. **执行检索**
   - 调用 `.agent/skills/00-系统核心/知识库管理/scripts/search_articles.py`
   - 传递关键词和检索参数

2. **处理结果**
   - 返回最相关的5篇文章
   - 包含文章路径、标题、相关度评分
   - 结果记录到 `00_素材索引.md`

**说明**：
- 会返回最相关的5篇文章
- 包含文章路径、标题、相关度评分
- 结果记录到 `00_素材索引.md`

### 网络搜索模式

1. **执行搜索**
   - 使用WebSearch工具搜索关键词
   - 获取最新的互联网信息

2. **显示结果**
   - 搜索结果摘要
   - 来源链接（markdown格式）
   - 按相关度排序

## 核心优势

- **统一入口**：一个命令完成所有检索需求
- **智能分层**：本地优先，向量库补充，网络兜底
- **多源整合**：整合本地、向量库、网络三种信息源
- **灵活选择**：支持指定检索方式或智能自动选择
- **结果标注**：清晰标注信息来源，便于追溯

## 技术实现

### 本地检索
- 使用Grep工具进行全文搜索
- 支持正则表达式匹配
- 自动过滤系统文件和临时文件

### 向量库检索
- 调用 `.agent/skills/00-系统核心/知识库管理/scripts/search_articles.py` 进行检索
- 使用 sentence_transformers 模型进行语义匹配
- 返回最相关的5篇文章

### 网络搜索
- 使用WebSearch工具
- 自动提取搜索结果和来源链接
- 返回markdown格式的结果
