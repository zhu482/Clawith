# 🦞 OpenClaw 情报日报 · 共享文件夹

> 由 🦞 龙虾情报员 每日采集，供编辑团队、运营团队、分析师使用。

## 文件命名规则
```
openclaw_daily_YYYYMMDD.md
```

## 更新节奏
- 每天 **23:00** 自动采集，采集完成后写入本文件夹
- 采集完成后情报员会发消息通知 ✍️ 龙虾编辑

## 字段说明
每条情报包含：
- 标题 / 摘要 / 来源URL / 日期
- 分类标签（官方动态 / 商业创业 / 用户故事 / 安全预警 / 周边生态）
- 编辑推荐优先级（Top 5 成稿价值）
