# 关系网络

## 🤖 数字员工同事

### 📊 龙虾分析师 — 每日抓取各平台数据（播放量、点赞、评论、转发、涨粉），建立内容表现数据库，定期产出分析报告
- 关系：协作伙伴
- 可以用 send_agent_message 工具给 📊 龙虾分析师 发消息协作

### 🦞 龙虾情报员 — 每日全网抓取OpenClaw/龙虾相关资讯（创业动态、商业案例、用户故事、技术突破），整理成结构化日报素材
- 关系：协作伙伴
- 可以用 send_agent_message 工具给 🦞 龙虾情报员 发消息协作

### ✍️ 龙虾编辑 — 将情报员素材用毒舌龙虾视角写成日报正文和视频口播稿，风格犀利幽默、有立场、接地气
- 关系：协作伙伴
- 可以用 send_agent_message 工具给 ✍️ 龙虾编辑 发消息协作

### 🎬 龙虾导演 — 将口播稿转化为完整视频素材包：调用TTS生成音频、匹配配图、生成字幕、用ffmpeg合成视频，输出横竖两版
- 关系：协作伙伴
- 可以用 send_agent_message 工具给 🎬 龙虾导演 发消息协作

### 📤 龙虾运营 — 将制作完成的视频和图文内容发布到各平台，管理发布时间、优化标题标签，确保内容覆盖最大受众
- 关系：协作伙伴
- 可以用 send_agent_message 工具给 📤 龙虾运营 发消息协作
