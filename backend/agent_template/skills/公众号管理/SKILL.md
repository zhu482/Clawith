# 公众号 - 公众号管理工具

统一的公众号文章下载、追更、清理工具。整合了下载、追更、清理三大功能。

## 使用方法

### 1. 追更所有关注的账号
```
/公众号 追更
```
自动追更AI和写作分类的所有关注账号，增量下载新文章。

### 2. 下载指定公众号
```
/公众号 下载 [公众号名称]
```
下载指定公众号的最新30篇文章。

### 3. 清理广告文章
```
/公众号 清理
```
自动清理知识库中的广告文章和无效文件。

### 4. 查看状态
```
/公众号 状态
```
查看API密钥状态、关注列表和文章统计。

### 5. 生成分析报告
```
/公众号 报告
```
生成公众号内容深度分析报告，包括基础统计、选题分析、文章索引等。

## 执行步骤

### 追更模式

1. **检查API密钥**
   - 检查密钥是否存在和过期
   - 如果过期，提示用户更新

2. **加载关注列表**
   - 从 `高质样例/公众号/_系统/关注列表.md` 读取
   - 支持AI和写作两个分类

3. **批量下载**
   - 遍历所有关注的账号
   - 搜索公众号获取fakeid
   - 下载最新文章（最多30篇）
   - 自动去重，跳过已存在的文章

4. **显示结果**
   - 新增文章数量
   - 跳过文章数量
   - 有更新的账号列表

### 下载模式

1. **查找分类**
   - 从关注列表中查找账号所属分类
   - 如果不在列表中，询问用户选择分类（AI/写作）

2. **下载文章**
   - 搜索公众号
   - 下载最新30篇文章
   - 自动去重

3. **显示结果**
   - 新下载文章数量
   - 跳过文章数量
   - 保存位置

### 清理模式

1. **遍历文章**
   - 遍历AI和写作两个分类下的所有文章
   - 跳过系统文件(_系统目录)

2. **检测广告**
   - 标题检测：包含广告关键词
   - 内容检测：文件太小(<1KB)或包含多个广告关键词

3. **删除文件**
   - 删除检测到的广告文章
   - 显示删除原因

4. **显示结果**
   - 删除文章数量
   - 保留文章数量

### 状态模式

1. **API密钥状态**
   - 检查密钥是否存在
   - 检查密钥是否过期（4天有效期）
   - 显示剩余有效天数

2. **关注列表**
   - 显示AI和写作分类的账号数量
   - 列出所有关注的账号

3. **文章统计**
   - 统计AI和写作分类的文章数量

### 报告生成模式

1. **数据收集**
   - 运行 `.agent/skills/05-内容工具/公众号管理/scripts/analyze_articles.py` 收集文章元数据
   - 统计文章数量、字数、标题特征等

2. **AI深度分析**
   - 读取统计数据
   - 分析选题类型、创新度、信息差策略
   - 识别热点事件和趋势

3. **生成报告**
   - 按照分析框架生成完整报告
   - 包含三大部分：基础统计、选题分析、文章索引

4. **保存报告**
   - 保存到 `高质样例/公众号/_系统/分析报告/`
   - 文件名格式：`周报_年份W周数_分析主题.md`

## 技术实现

### 脚本位置
- `.agent/skills/05-内容工具/公众号管理/scripts/wechat_manager.py` - 统一的公众号管理脚本
- `.agent/skills/05-内容工具/公众号管理/scripts/analyze_articles.py` - 文章数据收集脚本（用于报告生成）

### 调用方式
```bash
python3 .agent/skills/05-内容工具/公众号管理/scripts/wechat_manager.py sync          # 追更
python3 .agent/skills/05-内容工具/公众号管理/scripts/wechat_manager.py download 账号名  # 下载
python3 .agent/skills/05-内容工具/公众号管理/scripts/wechat_manager.py clean         # 清理
python3 .agent/skills/05-内容工具/公众号管理/scripts/wechat_manager.py status        # 状态
python3 .agent/skills/05-内容工具/公众号管理/scripts/analyze_articles.py             # 收集数据（报告生成第一步）
```

### 配置文件
- **API密钥**: `.wechat_api_config`
- **关注列表**: `高质样例/公众号/_系统/关注列表.md`
- **下载位置**: `高质样例/公众号/[分类]/[账号名]/`
- **统计数据**: `高质样例/公众号/_系统/文章统计数据.json`
- **分析报告**: `高质样例/公众号/_系统/分析报告/`

## 注意事项

### API密钥管理
- API密钥每4天过期一次
- 过期后需访问 https://down.mptext.top 获取新密钥
- 密钥配置格式: `API_KEY="your_key_here"`

### 下载规则
- 每次最多下载30篇文章
- 自动去重，跳过已存在的文章
- 文件名格式: `日期-标题.md` (如: `20260113-AI时代的能力.md`)
- 请求间隔0.5秒，避免过快

### 清理规则
- 删除操作不可恢复，请谨慎使用
- 建议定期清理，保持知识库质量
- 广告关键词可在脚本中自定义

### 报告生成规则
- 报告生成分两步：先收集数据，再AI分析
- 数据收集脚本会统计所有文章的元数据
- AI分析需要我(Claude)参与，提供深度洞察
- 报告参考框架：`高质样例/公众号/_系统/分析报告/` 下的示例
- 建议定期生成(如每周或每月)，跟踪内容趋势

## 核心优势

- **统一管理**：一个工具完成所有公众号相关操作
- **智能去重**：自动跳过已下载的文章
- **分类管理**：支持AI和写作两个分类
- **自动清理**：智能识别并清理广告文章
- **状态监控**：实时查看系统状态和统计信息
- **深度分析**：AI驱动的内容分析报告，洞察选题趋势和创新方向
